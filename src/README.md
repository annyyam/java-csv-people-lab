# Java CSV People Lab

Лабораторная работа 4 по Java.

## Описание

Программа читает CSV-файл с информацией о людях и преобразует данные в список объектов `Person`.

Каждый человек содержит:

- ID
- имя
- пол
- дату рождения
- подразделение
- зарплату

Подразделение является отдельной сущностью `Department` с собственным ID и названием.

## Технологии

- Java 17
- Maven
- OpenCSV
- JUnit 5
